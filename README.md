## Get started

1. Install dependencies

   ```bash
   yarn
   ```
2. Install cocoapods

   ```bash
    cd ios
   ```

   ```bash
    pod install
   ```
3. Create .env file and add required keys

   ```bash
      SL_SDK_API_KEY = "YOUR SDK KEY"
      SL_BRANCH_API_KEY = "YOUR BRANCH KEY"
   ```

4. Start the app

## Android

### Run metro 

   ```bash
    yarn start
   ```

### Press "a" to run on Android


## iOS

### Open Xcode
